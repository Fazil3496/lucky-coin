# 🍀 Lucky Coin

Lucky Coin is a fun web app that helps users discover their daily fortune through a virtual coin flip.

## Features

* 🎲 Coin Flip Fortune System
* 🍀 Daily Mood Prediction
* 🌟 Daily Quotes
* 🎯 Daily Challenges
* 📜 Flip History
* 📤 Share Results
* 🔥 Daily Streak Tracking
* 🏆 Achievements System
* 🎨 Dark / Light Theme
* ❓ FAQ Section
* 🎡 Lucky Wheel
* ❤️ Motivation Buddy
* ⚙️ Settings Panel

## Technologies Used

* HTML
* CSS
* JavaScript
* Local Storage

## How It Works

Users can flip the coin up to 3 times per day and receive a fortune score, lucky number, lucky color, and personalized advice. Additional features like the Lucky Wheel, streaks, achievements, and Motivation Buddy make the experience more engaging.
